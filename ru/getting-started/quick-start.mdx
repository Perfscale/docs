## 1. Войдите в систему

Перейдите на [http://localhost:3000/login](http://localhost:3000/login) и войдите с демо-данными:

```
demo@perfscale.ru / demo1234
```

## 2. Проверьте машины

Вкладка **Машины** показывает все зарегистрированные генераторы нагрузки. В демо-workspace есть два онлайн-узла (`worker-node-01`, `worker-node-02`) и один офлайн Mac builder.

Машина готова к работе, когда её статус отображает **Подключена**.

## 3. Изучите пример теста

Перейдите на вкладку **Тесты**. Там вы найдёте *API Smoke Test*, настроенный для k6:

```javascript
import http from 'k6/http';
import { check, sleep } from 'k6';

export const options = {
  vus: 10,
  duration: '30s',
};

export default function () {
  const res = http.get('https://httpbin.org/get');
  check(res, { 'status is 200': (r) => r.status === 200 });
  sleep(1);
}
```

## 4. Запустите задачу

Нажмите **Запустить тест**, чтобы создать новую задачу. perfscale распределит её по доступным машинам и будет передавать метрики в реальном времени:

- Запросов в секунду
- Время ответа p50 / p95 / p99
- Процент ошибок

## 5. Просмотрите результаты

После завершения задачи вкладка **Запуски и логи** покажет полный итоговый отчёт. В демо-аккаунте есть одна завершённая задача с заранее заполненными метриками, которые можно изучить сразу.
